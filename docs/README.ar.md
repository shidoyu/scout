# scout — البحث على الويب وجلب المحتوى

تُعيد أداة WebSearch المدمجة في Claude Code مقتطفات مكونة من 125 حرفًا وتعتمد على مطابقة الكلمات المفتاحية فحسب. يحوّل scout السؤال إلى استعلامات محسّنة متعددة المحركات، ويقيّم جودة النتائج، ويُعيد البحث عند الحاجة — للوصول إلى المصادر الأولية بشكل أسرع وأكثر موثوقية.

## المميزات

- **scout:search** — بحث ويب متعدد المحركات مع تحسين تصميم الاستعلامات
- **scout:fetch** — جلب محتوى الروابط مع اختيار الأداة المناسبة مراعاةً للخصوصية

## التثبيت

```bash
claude plugin add shidoyu/scout
```

## البدء السريع

يعمل scout فور التثبيت — يستخدم البحث كلًا من WebSearch (المدمج) و Exa (مجاني، لا يحتاج مفتاح API). الإعداد الاختياري يضيف مزيدًا من الإمكانيات:

```bash
bash tools/setup.sh
```

## المهارات

### scout:search

بحث ويب ذكي يتضمن:
- بحثًا أوليًا لتحسين الاستعلامات
- تصميم استعلامات متعددة اللغات
- محركات بحث متعددة (WebSearch، البحث الدلالي عبر [Exa](https://exa.ai))
- HyDE ([تضمينات المستندات الافتراضية](https://arxiv.org/abs/2212.10496)) للاستعلامات المفاهيمية عبر Exa
- تقييم الجودة مع حلقة إعادة بحث تلقائية

الاستخدام: `/scout:search سؤالك هنا`

### scout:fetch

جلب محتوى صفحات الويب مع تصنيف الخصوصية التلقائي:
- **الصفحات العامة** → Jina Reader (يتطلب مفتاح API) / WebFetch (بديل مدمج)
- **الصفحات السرية** → Playwright المحلي (بدون استدعاءات API خارجية)
- **الصفحات التي تتطلب مصادقة** → Chrome DevTools (جلسة المتصفح)

الاستخدام: `/scout:fetch URL`

## الإعداد (اختياري)

شغّل `tools/setup.sh` لضبط الإعدادات:

1. **Exa** — أدوات بحث متقدمة مدعومة بالذكاء الاصطناعي (مفتاح API للميزات المدفوعة؛ الفئة المجانية تعمل بدون إعداد)
2. **Jina Reader** — جلب صفحات ويب عالي الجودة بصيغة Markdown (يتطلب مفتاح API؛ بدونه تعود الصفحات العامة إلى WebFetch)
3. **Playwright** — جلب قائم على المتصفح للصفحات المعتمدة على JavaScript والصفحات السرية (تنزيل بحجم ~200MB)

جميع الخطوات قابلة للتخطي. يمكن إعادة التشغيل في أي وقت لتحديث الإعدادات.
بعد الإعداد، أعد تشغيل Claude Code (أو شغّل `/mcp`) لتفعيل خوادم MCP الجديدة.

## الخصوصية

يصنّف scout الروابط إلى ثلاثة مستويات قبل الجلب:
- **عام** → واجهات برمجة سحابية (Jina Reader / WebFetch)
- **سري** → Playwright المحلي فقط (التوجيه المقصود: لا تُرسل الروابط السرية إلى واجهات برمجة خارجية)
- **يتطلب مصادقة** → Chrome DevTools (يستخدم جلسة المتصفح الخاصة بك)

هذا التصنيف تلقائي لكنه يستند إلى حكم نموذج اللغة، وليس ضمانًا مفروضًا على مستوى النظام. راجع [إخلاء مسؤولية الخصوصية](#إخلاء-مسؤولية-الخصوصية) للتفاصيل.

## المتطلبات

- Claude Code
- `jq` (لسكريبت الإعداد فقط)
- `npm`/`npx` (لخادم [MCP](https://modelcontextprotocol.io/): chrome-devtools)

- Chrome (اختياري، لجلب الصفحات التي تتطلب مصادقة عبر DevTools)

### إعداد Chrome DevTools (للصفحات التي تتطلب مصادقة)

لجلب الصفحات التي تتطلب تسجيل دخول (OAuth، لوحات تحكم SaaS)، يجب تشغيل Chrome في وضع التصحيح:

```bash
# macOS
open -a "Google Chrome" --args --remote-debugging-port=9222

# Linux
google-chrome --remote-debugging-port=9222
```

## إخلاء مسؤولية الخصوصية

يصنّف scout الروابط حسب حساسيتها ويوجّه الروابط السرية إلى أدوات محلية فقط.
يستند هذا التصنيف إلى حكم نموذج اللغة (أنماط النطاق والسياق) وهو **ليس ضمانًا مفروضًا على مستوى النظام**.
للبيانات شديدة الحساسية، تحقق من التصنيف قبل المتابعة.

**ملف تعريف المتصفح.** يستخدم أداة الجلب القائمة على Playwright (`fetch-page.py`) ملف تعريف متصفح مستمرّ (persistent) (`tools/.chrome-profile/`) قد يتراكم فيه ملفات تعريف الارتباط وبيانات الجلسة وسجل التصفح. هذا المجلد مستبعد من Git عبر `.gitignore` لكن قد تنسخه أدوات النسخ الاحتياطي أو خدمات المزامنة السحابية. احذف المجلد دوريًا إذا كنت تجلب صفحات سرية.

## إخلاء المسؤولية

يُقدَّم هذا البرنامج الإضافي "كما هو" بموجب رخصة MIT، دون أي ضمان من أي نوع.

**واجهات برمجة خارجية.** يعتمد هذا البرنامج الإضافي على واجهات برمجة تابعة لجهات خارجية (Exa و Jina AI وغيرها). لا يقدّم المؤلف أي ضمانات بشأن توفر هذه الخدمات أو دقتها أو أسعارها أو استمراريتها، وهو غير مسؤول عن أي تكاليف تنجم عن استخدام API.

**إدارة مفاتيح API.** أنت المسؤول الوحيد عن الحصول على مفاتيح API الخاصة بك وتأمينها وإدارتها، والالتزام بشروط خدمة كل مزوّد.

**تصنيف المحتوى.** عند جلب محتوى الويب، قد يستخدم البرنامج الإضافي تصنيفًا مستندًا إلى نموذج اللغة لتقييم حساسية الخصوصية وتحديد أساليب الاسترداد المناسبة. هذه التصنيفات تبذل جهدًا أقصى وقد تحتوي على أخطاء. لا تعتمد على التصنيف التلقائي كضمانة وحيدة للمعلومات الحساسة أو السرية.

**جلب الويب وأتمتة المتصفح.** يتضمن هذا البرنامج الإضافي أدوات لأتمتة المتصفح بدون واجهة رسومية عبر Playwright و Chrome DevTools. أنت مسؤول عن ضمان توافق استخدامك مع شروط خدمة المواقع المستهدفة وسياسات robots.txt والقوانين المعمول بها. المؤلف غير مسؤول عن حظر المواقع أو تعليق الحسابات أو قيود IP أو تنفيذ سكريبتات غير متوقع أو استهلاك الموارد أو مشاكل التوافق الناجمة عن أتمتة المتصفح.

**خوادم MCP.** يتصل هذا البرنامج الإضافي بخوادم MCP (بروتوكول سياق النموذج) تابعة لجهات خارجية. لا يتحكم المؤلف في سلوك هذه الخوادم أو أمانها ولا يراجعها أو يضمنها.
