# scout — חיפוש ברשת ואחזור תוכן

החיפוש המובנה WebSearch של Claude Code מחזיר קטעים של 125 תווים ומסתמך על התאמת מילות מפתח בלבד. scout הופך שאלה מעורפלת לשאילתות מותאמות לכמה מנועי חיפוש, מעריך את איכות התוצאות ומבצע חיפוש חוזר לפי הצורך — ומגיע למקורות ראשוניים מהר ובאמינות רבה יותר.

## תכונות

- **scout:search** — חיפוש ברשת עם מספר מנועים ואופטימיזציה של עיצוב שאילתות
- **scout:fetch** — אחזור תוכן מ-URL עם בחירת כלי המתחשבת בפרטיות

## התקנה

הריצו בטרמינל:

```bash
# שלב 1: רישום ה-marketplace
claude plugin marketplace add shidoyu/scout

# שלב 2: התקנת התוסף
claude plugin install scout@shidoyu-scout
```

## התחלה מהירה

scout פועל מיד לאחר ההתקנה — החיפוש משתמש ב-WebSearch (מובנה) וב-Exa (חינמי, ללא צורך במפתח). הגדרה נוספת (אופציונלית) מרחיבה את היכולות:

```bash
bash tools/setup.sh
```

## Skills

### scout:search

חיפוש ברשת אינטליגנטי הכולל:
- מחקר מקדים לשיפור השאילתות
- עיצוב שאילתות בריבוי שפות
- מספר מנועי חיפוש (WebSearch, חיפוש סמנטי של [Exa](https://exa.ai))
- HyDE ([Hypothetical Document Embeddings](https://arxiv.org/abs/2212.10496)) לשאילתות מושגיות דרך Exa
- הערכת איכות עם לולאת חיפוש חוזר אוטומטית

שימוש: `/scout:search השאלה שלך כאן`

### scout:fetch

אחזור תוכן דף אינטרנט עם סיווג פרטיות אוטומטי:
- **דפים ציבוריים** → Jina Reader (נדרש מפתח API) / WebFetch (חלופה מובנית)
- **דפים חסויים** → Playwright מקומי (ללא קריאות API חיצוניות)
- **דפים מאומתים** → Chrome DevTools (סשן דפדפן)

שימוש: `/scout:fetch URL`

## הגדרה (אופציונלית)

הרץ את `tools/setup.sh` להגדרת:

1. **Exa** — כלי חיפוש מתקדמים מבוססי AI (מפתח API לתכונות בתשלום; רמה חינמית פועלת ללא הגדרה)
2. **Jina Reader** — אחזור דפי אינטרנט באיכות גבוהה כ-Markdown (נדרש מפתח API; ללא מפתח, דפים ציבוריים חוזרים ל-WebFetch)
3. **Playwright** — אחזור מבוסס דפדפן לדפים עם JavaScript ודפים חסויים (הורדה של ~200MB)

אפשר לדלג על כל שלב. ניתן להריץ מחדש בכל עת לעדכון הגדרות.
לאחר ההגדרה, הפעל מחדש את Claude Code (או הרץ `/mcp`) כדי ששרתי MCP חדשים ייכנסו לתוקף.

## פרטיות

scout מסווג כתובות URL לשלוש רמות לפני האחזור:
- **ציבורי** → ממשקי API בענן (Jina Reader / WebFetch)
- **חסוי** → Playwright מקומי בלבד (ניתוב מכוון: כתובות URL חסויות אינן נשלחות לממשקי API חיצוניים)
- **מאומת** → Chrome DevTools (משתמש בסשן הדפדפן שלך)

סיווג זה הוא אוטומטי אך מבוסס על שיפוט LLM, לא על אכיפת מערכת. לפרטים נוספים ראו את סעיף [כתב הוויתור על פרטיות](#כתב-ויתור-על-פרטיות).

## דרישות

- Claude Code
- `jq` (לסקריפט ההגדרה בלבד)
- `npm`/`npx` (לשרת [MCP](https://modelcontextprotocol.io/): chrome-devtools)
- Python 3.10+ (אופציונלי, לאחזור מקומי עם Playwright)
- `uvx` או `uv` (אופציונלי, לשרת MCP: markitdown — המרת HTML ל-Markdown)
- Chrome (אופציונלי, לאחזור דפים מאומתים דרך DevTools)

### הגדרת Chrome DevTools (לדפים מאומתים)

לאחזור דפים הדורשים התחברות (OAuth, לוחות מחוונים של SaaS), Chrome חייב לפעול במצב ניפוי שגיאות:

```bash
# macOS
open -a "Google Chrome" --args --remote-debugging-port=9222

# Linux
google-chrome --remote-debugging-port=9222
```

## כתב ויתור על פרטיות

scout מסווג כתובות URL לפי רגישות ומנתב כתובות URL חסויות לכלים מקומיים בלבד.
סיווג זה מבוסס על שיפוט LLM (תבניות דומיין והקשר) ו**אינו מהווה אכיפה מצד המערכת**.
עבור נתונים רגישים ביותר, אמת את הסיווג לפני המשך.

**פרופיל דפדפן.** מאחזר הדפים המבוסס על Playwright (`fetch-page.py`) משתמש בפרופיל דפדפן מתמיד (`tools/.chrome-profile/`) שעשוי לצבור עוגיות, נתוני סשן והיסטוריית גלישה. ספרייה זו אינה נכללת ב-Git דרך `.gitignore` אך עשויה להיות מועתקת על ידי כלי גיבוי או שירותי סנכרון בענן. מחק את הספרייה מעת לעת אם אתה מאחזר דפים חסויים.

## כתב ויתור

פלאגין זה מסופק "כמות שהוא" תחת רישיון MIT, ללא כל אחריות.

**ממשקי API חיצוניים.** פלאגין זה מסתמך על ממשקי API של צד שלישי (Exa, Jina AI ואחרים). המחבר אינו מעניק כל ערבות לגבי הזמינות, הדיוק, התמחור או הרציפות של שירותים אלו ואינו אחראי לעלויות שנגרמו משימוש בממשק API.

**ניהול מפתחות API.** אתה האחראי הבלעדי לקבלת, אבטחת וניהול מפתחות ה-API שלך, ולציות לתנאי השירות של כל ספק.

**סיווג תוכן.** בעת אחזור תוכן אינטרנט, הפלאגין עשוי להשתמש בסיווג מבוסס LLM להערכת רגישות פרטיות וקביעת שיטות אחזור מתאימות. סיווגים כאלה הם מאמץ מיטבי ועשויים להכיל שגיאות. אל תסתמך על סיווג אוטומטי כאמצעי ההגנה היחיד על מידע רגיש או חסוי.

**אחזור אינטרנט ואוטומציה של דפדפן.** פלאגין זה כולל כלים לאוטומציה של דפדפן headless דרך Playwright ו-Chrome DevTools. אתה אחראי להבטיח שהשימוש שלך תואם לתנאי השירות של אתרי היעד, מדיניות robots.txt וחוקים החלים. המחבר אינו אחראי לחסימת אתר, השעיית חשבון, הגבלות IP, ביצוע סקריפטים בלתי צפוי, צריכת משאבים או בעיות תאימות הנובעות מאוטומציה של דפדפן.

**שרתי MCP.** פלאגין זה מתחבר לשרתי MCP (Model Context Protocol) של צד שלישי. המחבר אינו שולט, מבקר או מתחייב לגבי ההתנהגות או האבטחה של שרתים אלו.
