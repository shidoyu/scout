# scout — Web Search & Content Fetching

WebSearch ที่ติดมากับ Claude Code นั้นคืนผลลัพธ์เป็นตัวอย่างข้อความความยาวเพียง 125 ตัวอักษร และอาศัยการจับคู่คำสำคัญเพียงอย่างเดียว scout แปลงคำถามที่กว้างๆ ให้กลายเป็น query ที่ปรับแต่งอย่างเหมาะสมสำหรับหลายเครื่องมือค้นหา ประเมินคุณภาพของผลลัพธ์ และค้นหาซ้ำเมื่อจำเป็น เพื่อเข้าถึงแหล่งข้อมูลหลักได้เร็วและน่าเชื่อถือยิ่งขึ้น

## ฟีเจอร์

- **scout:search** — การค้นหาเว็บแบบหลายเครื่องมือพร้อมการออกแบบ query ที่ปรับแต่งอย่างชาญฉลาด
- **scout:fetch** — การดึงเนื้อหาจาก URL พร้อมการเลือกใช้เครื่องมือที่คำนึงถึงความเป็นส่วนตัว

## การติดตั้ง

```bash
claude plugin add shidoyu/scout
```

## เริ่มต้นใช้งานอย่างรวดเร็ว

scout พร้อมใช้งานทันทีหลังติดตั้ง โดยการค้นหาใช้ WebSearch (ติดมาในตัว) และ Exa (ฟรี ไม่ต้องใช้ API key) การตั้งค่าเพิ่มเติมจะเปิดใช้งานความสามารถอื่น:

```bash
bash tools/setup.sh
```

## Skills

### scout:search

การค้นหาเว็บอัจฉริยะที่มี:
- การค้นคว้าเบื้องต้นเพื่อปรับแต่ง query
- การออกแบบ query หลายภาษา
- เครื่องมือค้นหาหลายตัว (WebSearch, [Exa](https://exa.ai) semantic search)
- HyDE ([Hypothetical Document Embeddings](https://arxiv.org/abs/2212.10496)) สำหรับ query เชิงแนวคิดผ่าน Exa
- การประเมินคุณภาพพร้อม auto re-search loop

การใช้งาน: `/scout:search คำถามของคุณที่นี่`

### scout:fetch

ดึงเนื้อหาหน้าเว็บพร้อมการจำแนกความเป็นส่วนตัวอัตโนมัติ:
- **หน้าสาธารณะ** → Jina Reader (ต้องใช้ API key) / WebFetch (ใช้เป็นทางเลือกสำรอง)
- **หน้าที่เป็นความลับ** → Playwright ในเครื่อง (ไม่เรียก API ภายนอก)
- **หน้าที่ต้องการการยืนยันตัวตน** → Chrome DevTools (ใช้ session ของเบราว์เซอร์)

การใช้งาน: `/scout:fetch URL`

## การตั้งค่า (ไม่บังคับ)

รัน `tools/setup.sh` เพื่อกำหนดค่า:

1. **Exa** — เครื่องมือค้นหาแบบ AI-native ขั้นสูง (ต้องใช้ API key สำหรับฟีเจอร์แบบชำระเงิน; ระดับฟรีใช้งานได้โดยไม่ต้องตั้งค่า)
2. **Jina Reader** — การดึงหน้าเว็บคุณภาพสูงเป็น Markdown (ต้องใช้ API key; หากไม่มี หน้าสาธารณะจะถูกเปลี่ยนไปใช้ WebFetch แทน)
3. **Playwright** — การดึงข้อมูลผ่านเบราว์เซอร์สำหรับหน้าที่แสดงผลด้วย JavaScript และหน้าที่เป็นความลับ (ดาวน์โหลดประมาณ ~200MB)

ทุกขั้นตอนสามารถข้ามได้ รันซ้ำได้ทุกเมื่อเพื่ออัปเดตการตั้งค่า
หลังการตั้งค่า ให้รีสตาร์ท Claude Code (หรือรัน `/mcp`) เพื่อให้ MCP server ใหม่ทำงาน

## ความเป็นส่วนตัว

scout จำแนก URL เป็นสามระดับก่อนดึงข้อมูล:
- **สาธารณะ** → Cloud APIs (Jina Reader / WebFetch)
- **ความลับ** → Playwright ในเครื่องเท่านั้น (routing ที่ตั้งใจ: URL ที่เป็นความลับจะไม่ถูกส่งไปยัง external API)
- **ต้องการยืนยันตัวตน** → Chrome DevTools (ใช้ session เบราว์เซอร์ของคุณ)

การจำแนกนี้เป็นอัตโนมัติแต่อาศัยการตัดสินของ LLM ไม่ใช่ system-level enforcement ดูรายละเอียดได้ที่[ข้อจำกัดความรับผิดชอบด้านความเป็นส่วนตัว](#ข้อจำกัดความรับผิดชอบด้านความเป็นส่วนตัว)

## ข้อกำหนด

- Claude Code
- `jq` (สำหรับสคริปต์ตั้งค่าเท่านั้น)
- `npm`/`npx` (สำหรับ [MCP](https://modelcontextprotocol.io/) server: chrome-devtools)
- Python 3.10+ (ไม่บังคับ สำหรับ Playwright local fetching)
- `uvx` หรือ `uv` (ไม่บังคับ สำหรับ MCP server: markitdown — การแปลง HTML→Markdown)
- Chrome (ไม่บังคับ สำหรับการดึงหน้าที่ต้องยืนยันตัวตนผ่าน DevTools)

### การตั้งค่า Chrome DevTools (สำหรับหน้าที่ต้องยืนยันตัวตน)

เพื่อดึงหน้าที่ต้องการเข้าสู่ระบบ (OAuth, SaaS dashboards) Chrome จะต้องรันในโหมด debug:

```bash
# macOS
open -a "Google Chrome" --args --remote-debugging-port=9222

# Linux
google-chrome --remote-debugging-port=9222
```

## ข้อจำกัดความรับผิดชอบด้านความเป็นส่วนตัว

scout จำแนก URL ตามความละเอียดอ่อนและกำหนดเส้นทาง URL ที่เป็นความลับไปยังเครื่องมือที่ทำงานในเครื่องเท่านั้น
การจำแนกนี้อาศัยการตัดสินของ LLM (รูปแบบโดเมนและบริบท) และ**ไม่ใช่การรับประกันที่บังคับใช้โดย system-level enforcement**
สำหรับข้อมูลที่ละเอียดอ่อนสูง โปรดตรวจสอบการจำแนกก่อนดำเนินการต่อ

**Browser Profile.** ตัวดึงข้อมูลที่ใช้ Playwright (`fetch-page.py`) ใช้ browser profile แบบถาวร (`tools/.chrome-profile/`) ซึ่งอาจสะสม cookies, ข้อมูล session และประวัติการท่องเว็บ ไดเรกทอรีนี้ถูกยกเว้นจาก Git ผ่าน `.gitignore` แต่อาจถูกคัดลอกโดยเครื่องมือสำรองข้อมูลหรือบริการ cloud sync ลบไดเรกทอรีนี้เป็นระยะหากคุณดึงข้อมูลหน้าที่เป็นความลับ

**อย่า commit `.mcp.json` ไปยัง Git** ใช้ `.mcp.json.dist` เป็นแม่แบบสำหรับการเผยแพร่

## ข้อจำกัดความรับผิดชอบ

ปลั๊กอินนี้ให้บริการ "as is" ภายใต้ MIT License โดยไม่มีการรับประกันใดๆ

**External APIs.** ปลั๊กอินนี้อาศัย API ของบุคคลที่สาม (Exa, Jina AI และอื่นๆ) ผู้เขียนไม่รับประกันเกี่ยวกับความพร้อมใช้งาน ความถูกต้อง ราคา หรือความต่อเนื่องของบริการเหล่านี้ และไม่รับผิดชอบต่อค่าใช้จ่ายที่เกิดขึ้นจากการใช้งาน API

**API Key Management.** คุณมีหน้าที่รับผิดชอบเองในการได้รับ รักษาความปลอดภัย และจัดการ API key ของคุณเอง รวมถึงการปฏิบัติตามข้อกำหนดการให้บริการของผู้ให้บริการแต่ละราย

**Content Classification.** เมื่อดึงเนื้อหาจากเว็บ ปลั๊กอินอาจใช้การจำแนกประเภทด้วย LLM เพื่อประเมินความละเอียดอ่อนด้านความเป็นส่วนตัวและกำหนดวิธีการดึงข้อมูลที่เหมาะสม การจำแนกดังกล่าวเป็นความพยายามที่ดีที่สุดและอาจมีข้อผิดพลาดได้ อย่าอาศัยการจำแนกอัตโนมัติเป็นมาตรการป้องกันเพียงอย่างเดียวสำหรับข้อมูลที่ละเอียดอ่อนหรือเป็นความลับ

**Web Fetching & Browser Automation.** ปลั๊กอินนี้มีเครื่องมือสำหรับการทำ headless browser automation ผ่าน Playwright และ Chrome DevTools คุณมีหน้าที่รับผิดชอบในการตรวจสอบให้แน่ใจว่าการใช้งานของคุณเป็นไปตามข้อกำหนดการให้บริการของเว็บไซต์เป้าหมาย นโยบาย robots.txt และกฎหมายที่บังคับใช้ ผู้เขียนไม่รับผิดชอบต่อการถูกบล็อกจากเว็บไซต์ การระงับบัญชี การจำกัด IP การรันสคริปต์โดยไม่คาดหมาย การใช้ทรัพยากร หรือปัญหาความเข้ากันได้ที่เกิดจากการทำ browser automation

**MCP Servers.** ปลั๊กอินนี้เชื่อมต่อกับ MCP (Model Context Protocol) server ของบุคคลที่สาม ผู้เขียนไม่ควบคุม ตรวจสอบ หรือรับประกันพฤติกรรมหรือความปลอดภัยของ server เหล่านี้
